== Description ==
Remove Website Field from Comment Form is the simplest way to remove website field from comment form. This is light & easy plugin. No settings. No extra code. Plug & play works. 
Enjoy :-)


== Installation ==

#### Manually
1. Download plugin archive.
1. Navigate to Plugins > Add New.
1. Click the Upload Plugin button at the top of the screen.
1. Select the zip archive from your local filesystem.
1. Click the Install Now button.
1. When the installation is complete, you’ll see “Plugin installed successfully.” Click the Activate Plugin button.
1. Enjoy :-)
